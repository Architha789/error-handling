Error Handling

1) Write a JavaScript function that takes a number as a parameter and throws a custom 'Error' if the number is not an integer.

2) Write a JavaScript function that accepts two numbers as parameters and throws a custom 'Error' if the second number is zero.

3) Write a JavaScript function that takes a number as a parameter and throws a custom 'Error' if the number is negative.

4) Write a JavaScript function that takes an array as a parameter and throws a custom 'Error' if the array is empty.

5) Write a JavaScript function that takes a string as a parameter and throws a custom 'Error' if the string is empty.
